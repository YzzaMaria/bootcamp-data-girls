# Trilha Análise de Dados – Data Girls

Repositório criado para armazenar os exercícios, desafios e projetos desenvolvidos durante a Trilha Análise de Dados – Data Girls.

Objetivos

- Praticar Python para análise de dados;
- Aprender processos de ETL (Extração, Transformação e Carga);
- Desenvolver habilidades em manipulação e visualização de dados;
- Construir um portfólio de projetos em Ciência e Análise de Dados.

Conteúdo

- Exercícios de análise e exploração de dados
- Estatística Basica
- Manipulação de dados com Pandas
- limpeza e tratamento dos dados
- SQL intermediário(CTEs e otimizações)
- ETL
- Visualização de dados
- Projetos práticos
- criação de Dashboards
Projeto em destaque

Ao final da trilha, este repositório contará com um projeto final, reunindo os conhecimentos adquiridos ao longo do curso.

Tecnologias utilizadas

- Python
- SQL
- Pandas
- Matplotlib
- Jupyter Notebook / Google Colab
- Power BL
Status

🚧 Em desenvolvimento — novos exercícios e projetos serão adicionados conforme o progresso na trilha.
